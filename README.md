# Description

Vue/Nuxt application to log catch reports using Firebase as backend.

When registered and logged in you are able to add/edit/delete catch reports and filter/sort on certain criteria.

## Used Technologies

- Vue
- Nuxt
- Vuetify
- GSAP
- Firebase
